About

This LuaU Obfuscator protects Roblox scripts using a custom virtual machine (VM), XOR string encoding, and a substitution table. It ensures robust security while maintaining compatibility with Roblox's LuaU environment.

Features

  • Custom VM: Executes code with randomized opcodes.
  • XOR Encoding: Encrypts strings with a random key.
  • Substitution Table: 16x16 table for byte-level obfuscation.
  • Randomized Variables: Unique variable names to deter deobfuscation.
  • Roblox Compatible: Works seamlessly in Roblox.

Security

  • XOR-based string encryption with random keys.
  • 16x16 substitution table for byte substitution.
  • Randomized opcodes (PUSH, POP, ADD, etc.) for control flow obfuscation.
  • Variable name randomization to obscure code structure.

How to Use

  1. Paste your LuaU script in the text area above.
  2. Click the "Obfuscate" button.
  3. Click the "Download Obfuscated Code" button to save the result.
  4. Use the obfuscated code in your Roblox project.
